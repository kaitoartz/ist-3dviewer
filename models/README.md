# Modelos 3D

Coloca tus archivos de modelos 3D aquí.

## Formatos soportados:
- **GLB** (recomendado) - Formato binario de glTF
- **GLTF** - Formato JSON de glTF

## Cómo usar tu modelo:

1. Coloca tu archivo `.glb` o `.gltf` en esta carpeta
2. En `js/main.js`, descomenta la última línea y ajusta el nombre:
   ```javascript
   loadCustomModel('models/tu-modelo.glb');
   ```

## Dónde conseguir modelos 3D gratis:
- [Sketchfab](https://sketchfab.com/feed)
- [Poly Haven](https://polyhaven.com/models)
- [Free3D](https://free3d.com/)
